AutoCSer 是一个以高效率为目标向导的互联网应用开发框架，主要包括高性能服务端通讯 RPC 组件与 WEB 数据视图模板组件。

1. 开发效率
. 采用自动化编码方案，结合静态代码生成技术与运行时代码生成技术 Emit，在应用层消除业务逻辑无关代码。
. WEB 组件架构采用前后端一体数据视图模型，目标是在高性能的基础上实现统一规划全局数据定义，从根本上消除服务端冗余的数据定义与逻辑。
. WEB 组件前端模块集成支持声明组件模式，目标是采用组件自举模式消除前端非业务代码。

2. 运行效率
提供高性能的基础组件，主要包括以下组件：
. 支持 200W+/s 异步调用吞吐的 高性能服务端通讯 RPC 组件
. 支持 80W+/s 调用的可靠持久化内存数据库组件，其中包含支持 40W+/s 消费的消息组件
. ORM 操作支持成员绑定，在写操作事件中可以跟踪所有数据变化内容，利用该特性可将指定数据库数据镜像映射到内存自定义数据结构实现高性能读取数据操作
. 对于 27 种常用数据类型组合的随机对象，二进制数据序列化支持单线程 200W+/s 操作、JSON 序列化 支持单线程 100W+/s 操作
以上测试环境为 Win10 + i5 6400 + 8GB ，不是在 VS 里调试运行。

基础类库支持 .NET Standard / Core 2.0 ，Web 数据视图框架支持 .NET Core 3.1

Windows 支持 .NET Framework 4.5

. AutoCSer 项目是核心程序集，主要包括 RPC 组件 与 二进制 / JSON 数据序列化组件。

. AutoCSer.CodeGenerator 项目用于静态代码生成，WEB 组件数据视图模型、可靠持久化内存数据库组件 都需要使用它生成部分代码。

. ExtensionTools\AutoCSer.Extensions 项目包括一些非核心常用模块，比如 XML 数据序列化、.NET 程序集 XML 文档解析、字符串表达式解析、常用异步回调包装操作、静态变量内存监控扫描、二叉平衡树、字节匹配状态机等，大多数扩展应用组件都依赖该项目。

. ExtensionTools\AutoCSer.NetCoreWeb 项目支持前后端一体数据视图模型的WEB 组件，.html 与 .js 支持 include 嵌入子文件操作，支持 JSON API 控制器的调用监控、鉴权、参数约束、日志等操作。

. ExtensionTools\AutoCSer.ORM 项目是基于单表格操作的 ORM 组件，支持自定义业务数据类型约束，支持 Lambda 表达式查询操作，支持读写引用隔离以便于查找写操作用于排查错误，支持远程代理读取数据操作用于提升网络吞吐性能与降低数据库连接数。

. Application\AutoCSer.CommandService.StreamPersistenceMemoryDatabase 项目是基于文件流可靠持久化的高性能内存数据库存组件，支持自定义任意数据结构，支持 API 级别事务，支持异地实时备份，支持动态加载程序集修复或者新增 API 避免不必要的冷启动操作。非持久化模式则相当于缓存服务组件，该组件中集成内置有消息组件。

. Application\AutoCSer.CommandService.TimestampVerify 项目是基于时间戳与 MD5 的验证组件，可以保证每一个验证数据仅能使用一次，避免验证网络数据被窃听。

. Application\AutoCSer.CommandService.ServiceRegistry 项目是动态服务注册组件，该组件实时推送服务变更信息，用于服务发现与服务集群功能，其中的本地端口注册组件是服务器的必备组件。

. Application\AutoCSer.CommandService.ProcessGuard 项目是应用进程守护组件，用于应用进程意外关闭自动重启支持，是服务器的必备组件。

. Application\AutoCSer.CommandService.DeployTask 项目是项目发布工具组件，是服务器的必备组件。

. Application\AutoCSer.CommandService.DiskBlock 项目是磁盘块对象储存组件，用于持久化对象的读写操作，对于热点缓存数据具有去重功能，配合使用可以降低内存数据库存组件的内存使用量以提升数据量上限。

. Application\AutoCSer.CommandService.FileSynchronous 项目是基于文件更新时间标识的文件上传下载组件。

. Application\AutoCSer.CommandService.DistributedLock 项目是分布式锁组件。

. Application\AutoCSer.CommandService.ReverseLogCollection 项目是反向日志收集组件，用于利用家庭带宽收集服务日志。

. Application\AutoCSer.CommandService.DatabaseBackup 项目是简单的 MSSQL 数据库文件备份组件。

. TestCase\ExtensionTools\AutoCSer.RandomObject 项目用于生成随机对象，用于生成测试数据。

. TestCase\ExtensionTools\AutoCSer.FieldEquals 项目用于对象深度对比，用于检查测试数据的匹配状态。
